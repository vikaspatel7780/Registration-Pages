# React MongoDB Express Authentication

This project is a demonstration of a basic authentication system using React, MongoDB, Express, and Tailwind CSS.

## Features

- **Login Page**
- **Registration Page**
- **Authentication**
- **MongoDB Database**

## Technologies Used

- React
- MongoDB
- Express
- Tailwind CSS

## Folder Structure

- `client/`
- `server/`

## Backend API Routes

- **POST `/api/auth/register`**
- **POST `/api/auth/login`**

## Available Scripts

- `npm start`
- `npm run dev`
- `npm test`
- `npm run build`
