# React Login and Registration Pages

This project provides simple login and registration pages implemented using React, MongoDB, Express, and Tailwind CSS.

## Features

- **Login Page**
- **Registration Page**

## Technologies Used

- React
- MongoDB
- Express
- Tailwind CSS

## Folder Structure

- `client/`
- `server/`

## Backend API Routes

- **POST `/api/auth/register`**
- **POST `/api/auth/login`**

## Available Scripts

- `npm start`
- `npm run dev`
- `npm test`
- `npm run build`
